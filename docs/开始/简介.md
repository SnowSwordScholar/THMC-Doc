# ThroughHome 思柔之乡
---
## 关于
**思柔之乡(ThroughHome)-MC** 是一个由热爱Minecraft的玩家群体共同打造的Minecraft社区，目前以 1.26.2 纯净生存主服为核心，并保留 1.20.4 怀旧服作为存档观光。

## 目的
  打造一个公益、开放、自由、温馨的玩家社区

## 服务器架构
截至Wiki编辑时间*2026/05/24*，ThroughHome 现采用「主服 + 怀旧服」双服架构。

> **服务器连接地址不在本文档列出**——请使用思柔之乡账号登录 [皮肤站](https://mc.th-mc.cn)，在首页置顶公告中查看「思柔之乡服务器IP列表」（包含 Java 主服 / 樱花 Frp 备线 / 基岩版主服 / 怀旧服 的全部地址）。

### 主服 (Java + 基岩)
- 内核版本：**Paper 1.26.2**，单服结构
- 当前仅开放 <font color="green">**主世界（纯净生存）**</font>，地图已重置
- 通过 **Geyser + Floodgate** 支持基岩版玩家加入  
  ※ 基岩版首次加入需在 Java 端完成账号绑定，详见 [基岩版加入指南](/加入服务器/基岩版加入指南)
- 自研皮肤插件保证 Java / 基岩两端均可正常显示思柔之乡皮肤站的皮肤

### 怀旧服
- 内核版本：1.20.4，沿用 Waterfall 代理
- 作为<font color="grey">**存档观光服**</font>有限开放，仅保留主世界
- 不再持续更新，仅供怀旧浏览
- 不支持基岩版加入

> **2026/05 架构变更说明**  
> 经 5/22 - 5/23 玩家投票（票数 4 : 2 : 5），最终采用「开 1.26 新地图为主地图，旧地图作为怀旧服有限开」方案。原有星梦世界、起床战争等子服暂未在新架构中重启。
