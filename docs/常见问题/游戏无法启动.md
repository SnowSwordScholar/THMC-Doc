# 游戏无法启动

> 🚧 **本页内容建设中** — 这里会汇总玩家加入服务器或本地启动游戏时常见的故障排查方法。

---

## 先做这三步基础检查

1. **客户端版本是否对应？**
   - 主服：Minecraft Java 版 **1.26.2**
   - 怀旧服：Minecraft Java 版 **1.20.4**
   - 客户端版本与服务器不一致会被直接拒绝
2. **是否完成皮肤站外置登录配置？**
   - 不论正版/离线玩家，都必须挂载思柔之乡皮肤站才能加入服务器
   - 配置方法见 [下载客户端 → 我想用自己的客户端加入服务器](/加入服务器/下载客户端#我想用自己的客户端加入服务器)
3. **服务器地址是否最新？**
   - 服务器 IP 以皮肤站置顶公告「思柔之乡服务器IP列表」为准
   - 旧整合包内置地址已过时，详见 [下载客户端](/加入服务器/下载客户端) 顶部提示

## 常见报错与处理

### Java 版

- **缺少 .net 错误**：旧整合包压缩包内附带安装程序，安装后重启即可
- **Java 版本错误 / Java 找不到**：使用完整整合包（自带 Java 环境），或自行安装与服务器对应的 Java 版本
- **认证失败 / 无法外置登录**：检查认证服务器是否填写为 `https://mc.th-mc.cn/api/yggdrasil`
- **连接超时**：尝试皮肤站公告中的「樱花 Frp 备用线路」

### 基岩版

- **进入服务器后无法移动 / 无法说话**：未完成账号绑定，请按 [基岩版加入指南](/加入服务器/基岩版加入指南) 操作
- **版本不兼容**：将基岩版客户端更新到最新版
