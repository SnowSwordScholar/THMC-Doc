# 服务端插件

> 🚧 **本页内容建设中**
>
> 思柔之乡主服（Paper 1.26.2）使用了若干公开插件与自研插件，本页将陆续整理对玩家可见的命令与功能。

---

## 跨版本支持

| 插件 | 类型 | 功能 |
|------|------|------|
| **Geyser** | 公开 | 让基岩版客户端能连接 Java 版服务器 |
| **Floodgate** | 公开 | 处理基岩版玩家在 Java 服务器上的账号身份 |
| **自研 — 反滥用绑定** | 自研 | 拦截未绑定的基岩版玩家所有动作，强制走 [基岩版加入指南](/加入服务器/基岩版加入指南) 中的 `/linkaccount` 流程 |
| **自研 — 跨平台皮肤同步** | 自研 | 让 Java 版与基岩版均能正常显示思柔之乡皮肤站的皮肤 |

## 玩家可用命令

| 命令 | 说明 |
|------|------|
| `/linkaccount` | Java 端发起基岩账号绑定，详见 [基岩版加入指南](/加入服务器/基岩版加入指南) |
