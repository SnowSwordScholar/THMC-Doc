## 跨版本支持

| 插件 | 类型 | 功能 |
|------|------|------|
| **Geyser** | 公开 | 让基岩版客户端能连接 Java 版服务器 |
| **Floodgate** | 公开 | 处理基岩版玩家在 Java 服务器上的账号身份 |
| **SimpleVoice-Geyser THMC Fork** | 维护分支 | 让基岩版玩家通过网页接入 Simple Voice Chat，支持网页聊天和 Tab 在线提示 |
| **自研 — 反滥用绑定** | 自研 | 拦截未绑定的基岩版玩家所有动作，强制走 [基岩版加入指南](/加入服务器/基岩版加入指南) 中的 `/linkaccount` 流程 |
| **自研 — 跨平台皮肤同步** | 自研 | 让 Java 版与基岩版均能正常显示思柔之乡皮肤站的皮肤 |
| **自研 — 区块强加载** | 自研 | 允许玩家自行登记需要常驻加载的生电区块，并在登录时自动续期 |

## 玩家可用命令

| 命令 | 说明 |
|------|------|
| `/linkaccount` | Java 端发起基岩账号绑定，详见 [基岩版加入指南](/加入服务器/基岩版加入指南) |
| `/svg pswd` | 打开网页语音密码设置表单，详见 [网页语音聊天](/加入服务器/网页语音聊天) |
| `/svg pswd <密码>` | 直接设置网页语音密码，不推荐在公共环境使用 |
| `/cl add` | 登记并强加载当前所在区块 |
| `/cl add chunk <区块X> <区块Z> [世界名]` | 按区块坐标登记并强加载指定区块 |
| `/cl add block <X> <Z> [世界名]` | 按方块坐标登记并强加载指定位置所在区块 |
| `/cl add block <X> <Y> <Z> [世界名]` | 使用完整物理坐标登记并强加载指定位置所在区块 |
| `/cl remove` | 移除当前所在区块的强加载记录 |
| `/cl remove chunk <区块X> <区块Z> [世界名]` | 按区块坐标移除强加载记录 |
| `/cl remove block <X> <Z> [世界名]` | 按方块坐标移除强加载记录 |
| `/cl list [页码]` | 查看自己的强加载区块记录 |
| `/cl renew` | 手动恢复并刷新自己的强加载区块记录 |
| `/cl info` | 查看自己的强加载时间与区块数量上限 |

## 区块强加载

区块强加载用于生电机器、农场、仓储等需要在玩家离开后继续运行的设施。请只为确实需要常驻运行的机器登记区块，避免无意义地增加服务器负载。

默认规则：

- 每名玩家最多登记 **9 个区块**。
- 每个区块的强加载有效期为 **3 天**。
- 玩家登录服务器后，插件会自动恢复并刷新自己登记过的区块，有效期重新变为 3 天。
- 如果玩家超过 3 天没有登录，区块会停止强加载，但记录会保留；下次登录后会自动恢复。
- 过期记录仍然占用 9 个区块名额，需要用 `/cl remove` 主动移除后才会释放名额。

坐标说明：

- 「区块坐标」是 F3 调试信息里的 Chunk 坐标，或方块坐标除以 16 后向下取整得到的坐标。
- 「方块坐标 / 物理坐标」就是平时看到的 X、Y、Z 坐标。插件会自动换算成对应区块。
- 不写世界名时，默认使用你当前所在的世界。

常用示例：

```text
/cl add
/cl add chunk 12 -8
/cl add block 200 -120
/cl add block 200 64 -120 world
/cl list
/cl remove
```
