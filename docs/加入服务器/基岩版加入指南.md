# 基岩版加入指南

自 2026/05 主服升级至 **Paper 1.26.2** 起，思柔之乡通过 **Geyser + Floodgate** 原生支持基岩版玩家加入主服。同时，经自研皮肤插件适配，**Java 版与基岩版在服务器内均可正常显示思柔之乡皮肤站的皮肤**。

---

## ⚠️ 重要前提

基岩版玩家**直接连接服务器后，角色的所有动作会被自研反滥用插件限制**——你能进入服务器，但无法移动、交互、聊天等。

这是为了防止未经认证的基岩账号绕过白名单机制。**唯一正确的接入方式**是：先在 Java 端完成账号绑定，再用基岩版进入。

---

## 📌 服务器连接地址在哪里？

本指南**不直接列出服务器 IP**。请按以下步骤获取：

1. 浏览器打开 [https://mc.th-mc.cn](https://mc.th-mc.cn)，使用思柔之乡账号登录
2. 在首页查看**置顶公告**「**思柔之乡服务器IP列表**」
3. 公告中你会看到：
   - **Java 版主线路**（用于步骤 1 的 Java 端登录）
   - **Java 版樱花 Frp 备用线路**（主线路异常时使用）
   - **基岩版主线路**（含端口，用于步骤 3 的基岩版连接）
   - **怀旧服主线路**（不在本指南范围内）

后文出现的「Java 主服地址」「基岩版地址 + 端口」均请以公告为准。

## 绑定流程

### 准备
- 已完成 [申请白名单](/加入服务器/申请白名单)
- 至少有一台可运行 Java 版 Minecraft 1.26.2 的设备
- Java 版已配置思柔之乡皮肤站外置登录（参见 [下载客户端](/加入服务器/下载客户端)）
- 已从皮肤站置顶公告中查到 Java 主服地址、基岩版地址 + 端口

### 步骤

**1. 用 Java 版登录主服**

启动 Java 版 Minecraft，使用公告中的 **Java 版主线路**地址连接。如主线路无法连接，可改用**樱花 Frp 备用线路**。

**2. 在游戏内执行绑定指令**

进入服务器后，在聊天框输入：

```
/linkaccount
```

系统将返回一段**绑定代码**，请记下或截图保留。

**3. 切换到基岩版连接服务器**

打开 Minecraft 基岩版，添加服务器，**地址与端口请填入公告中的「基岩版主线路」**。

**4. 在基岩版输入绑定代码**

进入服务器后，按游戏内提示输入第 2 步获得的绑定代码，绑定即完成。

**5. 享受游戏**

绑定完成后，基岩版账号即可正常游玩，无需每次重新绑定。

## 关于皮肤

服务器内自研皮肤插件已打通 Java / 基岩两端，**只要你在思柔之乡皮肤站上传/选择了皮肤，两个版本下都会同步显示**，不再出现"基岩版看不到 Java 玩家皮肤"或反之的问题。

---

## 常见问题

**Q：我没有 Java 版，能直接用基岩版玩吗？**  
A：暂时不能。绑定流程必须在 Java 端发起。可以借朋友的电脑、或临时使用免费 Java 启动器完成一次绑定，后续就只用基岩版即可。

**Q：绑定代码失效了？**  
A：在 Java 端重新执行 `/linkaccount` 获取新代码。

**Q：连接基岩版时提示版本不兼容？**  
A：Geyser 通常会跟进当前主流基岩版本，请确保你的基岩版客户端为最新版。如仍有问题，请到官方 Q 群 (852509714) 反馈。

**Q：怀旧服支持基岩版吗？**  
A：怀旧服未部署 Geyser，仅支持 Java 版 1.20.4 进入。

**Q：登录皮肤站后没看到那条置顶公告？**  
A：刷新页面，或在公告板/通知中心查看。如仍看不到，请到官方 Q 群 (852509714) 联系管理员。
