# 下载 Minecraft 客户端

欢迎来到思柔之乡！  
**你可以使用任何你喜欢的 Minecraft 客户端加入服务器**——原版启动器、HMCL、PCL2、BakaXL、Prism Launcher、官方启动器，都没问题。我们对客户端没有限制，唯一的要求是：**不要恶意对待其他玩家**。

只需在你常用的启动器里完成两件事：

1. 选择和服务器对应的版本（主服 **1.26.2**，怀旧服 **1.20.4**）
2. 配置思柔之乡皮肤站的**外置登录**（详见下文）

> ## ⚠️ 重要提示（2026/05 更新）
> **本页底部的整合客户端尚未跟进 1.26.2 主服**，版本与当前主服不匹配，直接使用会无法连接主服。在新整合包发布前，请使用自己的启动器并参考本页指引配置。

> ### 📌 服务器连接地址说明
> 出于安全考虑，本文档不直接列出服务器 IP。  
> 请使用思柔之乡账号登录皮肤站 [https://mc.th-mc.cn](https://mc.th-mc.cn)，首页**置顶公告**中的「**思柔之乡服务器IP列表**」会列出：
> - Java 版主线路 / 樱花 Frp 备用线路
> - 基岩版主线路（含端口）
> - 怀旧服主线路
>
> 后文中提到的「主服地址」「怀旧服地址」「基岩版地址」均以该公告为准。

## 推荐的整合包与光影

如果你不想自己折腾性能优化，推荐使用以下配置（自由选择，并非必须）：

- **整合包**：[Remarkably Optimized](https://modrinth.com/modpack/remarkably) — 一个专注于性能优化的轻量整合包，几乎不改变原版体验，但帧数更高、加载更快。请选择与服务器对应的版本（主服 1.26.2 / 怀旧服 1.20.4）。
- **光影**：[Photon Shader](https://modrinth.com/shader/photon-shader) — 画面表现优秀且对硬件相对友好，与 Iris/Sodium 兼容良好。

> 提示：Remarkably Optimized 已内置 Iris/Sodium，安装好整合包后直接把 Photon 的 zip 文件丢进 `shaderpacks/` 目录即可启用。

## 配置外置登录（必做）

无论正版还是离线玩家，**加入思柔之乡都需要把启动器登录方式切换到思柔之乡皮肤站的外置登录**，否则无法进入服务器。

外置登录服务地址（认证服务器 / Yggdrasil API）：

```
https://mc.th-mc.cn/api/yggdrasil
```

### 方案一：把下面的按钮直接拖进启动器（推荐）

按住下面这个按钮，拖到正在运行的启动器窗口（HMCL、PCL2、BakaXL 等都支持），启动器会自动识别并配置好认证服务器，无需手动输入。

> 操作说明：**鼠标左键按住** 上面的按钮不松手，拖到 HMCL / PCL2 / BakaXL 等启动器的窗口里再松开，启动器会弹出对话框确认添加认证服务器，确认即可。
>
> 如果你的启动器不识别拖入（少数旧版或自定义启动器可能不支持），请使用下面的方案二或方案三。
>
> 也可在皮肤站 [https://mc.th-mc.cn](https://mc.th-mc.cn) 登录后从那里拖入，效果一致。

### 方案二：参考 Blessing Skin 官方指引手动配置

如果你的启动器无法识别拖入，或你想了解原理，可参考：

> Blessing Skin 官方教程：<https://blessing.netlify.app/yggdrasil-api/authlib-injector.html>

按教程把认证服务器填为 `https://mc.th-mc.cn/api/yggdrasil` 即可。

### 方案三：在启动器里手动填写（以 PCL2 为例）

如果你习惯在启动器里手动配置，以 **PCL2** 为例：

1. 选择和服务器匹配的游戏版本（主服 **1.26.2** / 怀旧服 **1.20.4**）
2. 依次点击 **版本设置** → **设置** → **服务器选项**，填入：
   - **登录方式（必选）**：`第三方登录：Authlib Injector 或 Little Skin`
   - **认证服务器（必填）**：`https://mc.th-mc.cn/api/yggdrasil`
   - **注册链接（必填）**：`https://mc.th-mc.cn`
   - **服务器名称**（选填）：`思柔之乡-ThroughHome`
   - **自动进入服务器**（选填）：地址请见皮肤站置顶公告中的「思柔之乡服务器IP列表」

> 基岩版玩家请参考 [基岩版加入指南](/加入服务器/基岩版加入指南)，需先在 Java 端完成账号绑定。
